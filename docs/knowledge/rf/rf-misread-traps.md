# RF Misread Traps

## 常见误读
- 把 Absolute Maximum 当 Recommended Operating Conditions。
- 只报 gain、NF、IIP3 的单个数字，不报对应模式和条件。
- 只看 marketing 文案里的最大输出，不看 ACLR/EVM 条件。
- 忽略 bypass、RX、TX、gain mode 切换差异。
- 忽略 layout、ground paddle、thermal via 和去耦要求。

- 哪几个图表最关键
- 哪些条件最容易误导新人
- 哪些地方决定能不能真正上板
