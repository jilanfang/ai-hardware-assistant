# RF Reading Method

## 阅读顺序
1. 首页、标题、feature list，确认器件定位。
2. block diagram，确认 TX/RX/旁路/检波/控制链。
3. 绝对最大额定值和推荐工作条件。
4. Electrical Characteristics，必须连同 test condition 一起看。
5. 典型曲线和表格，确认模式、调制、带宽和温度条件。
6. truth table、timing、control interface。
7. package、layout、thermal、application circuit。

## 必看提醒
- 不要把饱和输出功率当线性输出功率。
- 不要把 typical 当 min/max 保证值。
- 不要忽略 modulation、bandwidth、temperature、supply、bias 条件。
