# Archive Guide

This directory holds material that is still useful for context, but should not drive current product or engineering decisions.

Use it for:

- historical strategy discussions
- prototype-era plans and specs
- old task history snapshots
- documents kept for traceability after scope changes

Current subdirectories:

- [product](/Users/jilanfang/ai-hardware-assistant/docs/archive/product)
  - historical product specs that are no longer active source of truth
- [strategy](/Users/jilanfang/ai-hardware-assistant/docs/archive/strategy)
  - historical market, positioning, and planning discussions
- [superpowers](/Users/jilanfang/ai-hardware-assistant/docs/archive/superpowers)
  - archived implementation plans and design specs
- [task-history](/Users/jilanfang/ai-hardware-assistant/docs/archive/task-history)
  - older progress, findings, and task-plan snapshots
